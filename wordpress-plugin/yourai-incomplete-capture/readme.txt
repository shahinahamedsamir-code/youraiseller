=== YourAI Seller Connect ===
Requires: WooCommerce

WHAT IT DOES
Connects your WooCommerce store to YourAI Seller:
- Captures unfinished checkouts (name/phone/address/cart) into your Incomplete tab
  while the customer types — even if they never click "Place Order".
- Order Guard: blocks checkout for phone / IP / email on your Order Block List.

INSTALL (no coding)
1. In your YourAI Seller dashboard open: Integration → WooCommerce → "Integration
   URLs & Keys" and copy your Business ID (biz_...) and API Key (yai_...).
2. WordPress admin → Plugins → Add New → Upload Plugin → choose this .zip → Install.
3. Activate.
4. Settings → YourAI Capture → paste the Business ID + API Key → Save.
   (If you downloaded the plugin from your dashboard, these are already filled in.)

Done. Incomplete checkouts now appear in your YourAI Seller Incomplete tab.
